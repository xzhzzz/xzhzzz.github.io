8种js数据类型判断方法

发表于 2023-02-12 更新于 2023-02-17

1、constructor

  • constuctor 指向创建该实例对象的构造函数

注意 null 和 undefined 没有 constructor，以及 constructor 可以被改写

2、typeof

  • 只能识别基础类型和引用类型

注意：null、 NaN、 document.all 的判断

3、instanceof

  • 语法：obj instanceof Type
  • 功能：判断 obj 是不是 Type 类的实例，只可用来判断引用数据
  • 实现思路： Type 的原型对象是否是 obj 的原型链上的某个对象
  • 注意：右操作数必须是函数或者 class

4、isPrototypeof

  • 是否在实例对象的原型链上
  • 基本等同于 instanceof

5、Object.prototype.toString

  • 利用函数动态 this 的特性

6、鸭子类型检测

  • 检查自身属性的类型或者执行结果的类型
  • 通常作为候选方案
  • 例子：kindof 与 p-is-promise

7、Symbol.toStringTag

  • 原理：Object.prototype.toString 会读取该值
  • 适用场景：需自定义类型
  • 注意事项：兼容性

8、等比较

  • 原理：与某个固定值进行比较
  • 适用场景：undefined、 window、 document、 null 等
