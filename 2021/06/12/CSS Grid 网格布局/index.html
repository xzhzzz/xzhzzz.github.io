CSS Grid 网格布局

发表于 2021-06-12 更新于 2022-12-16

介绍

CSS Grid 网格布局（又称为 “Grid” ），是一个二维的基于网格的布局系统，它的目标是完全改变我们基于网格的用户界面的布局方式。

与flex布局的区别

flex布局是一维布局系统，CSS Grid 布局是一个二维布局系统，也就意味着它可以同时处理列和行，因此Grid 布局远比 Flex 布局强大。

通过将 CSS 规则应用于父元素 (成为 Grid Container 网格容器)和其子元素（成为 Grid Items 网格项），你就可以轻松使用 Grid布局。

术语

网格容器(Grid Container)和项目（Grid Item）

应用 display: grid 的元素。这是所有网格项（grid item）的直接父级元素。在下面例子中，container 就是 **网格容器(Grid Container)，item就是网格项目(Grid Item)**。

网格线(Grid Line)

构成网格结构的分界线。它们既可以是垂直的（列网格线(column grid lines)），也可以是水平的（行网格线(row grid lines)），并位于行或列的任一侧。

网络轨道(Grid Track)

可以理解为行和列所布局的方向，默认从左到右，从上到下。

网格单元(Grid Cell)

相邻行和相邻列网格线之间到空间。这是 Grid系统的一个“单元”。

网格区域(Grid Area)

4个网格线包围的总空间。网格区域可以包含任意数量的网格单元。

属性

  • grid-template-columns、grid-template-rows 控制行列数、宽高 demo地址
    • repeat()
    • auto-fill关键字
    • fr关键字
    • minmax()
    • auto关键字
  • grid-gap 控制每个单元的间距
  • grid-template-areas 网格布局允许指定”区域”（area），一个区域由单个或多个单元格组成。demo地址
  • grid-auto-flow 划分网格以后，容器的子元素会按照顺序，自动放置在每一个网格。默认的放置顺序是”先行后列”，即先填满第一行，再开始放入第二行。
  • justify-items、align-items、place-items 控制每个单元的位置 demo地址
    • start ｜ end ｜ center ｜stretch 拉伸填满
  • justify-content、align-content、place-content 控制整个内容区域在容器里面的位置
    • start | end | center | stretch | space-around | space-between | space-evenly
  • grid-auto-columns、grid-auto-rows 用于自动生成多余的网格 demo地址
  • grid-column-start、 grid-column-end、 grid-row-start、grid-row-end 用于指定单元的位置，具体方法就是指定项目的四个边框，分别定位在哪根网格线 demo地址
  • grid-column、 grid-row 是上面属性的合并简写属性 demo地址
    • grid-column：grid-column-start / grid-column-end
    • grid-row：grid-row-start / grid-row-end
  • grid-area 合并简写属性
    • grid-area： grid-row-start / grid-column-start / grid-row-end / grid-column-end
  • justify-self、align-self、place-self 设置单元格内容的位置
    • start | end | center | stretch
